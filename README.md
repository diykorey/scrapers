# scrapers

Install scrapy

    pip install Scrapy
    
Run scraper with command

    scrapy crawl djinnispider -o output.csv
    
    Where
        djinnispider - name of spider, also there is another spider for ilustrative purpose - loginspider
        output.csv - name of output file
